Sortierverfahren

Bubble Sort:

Beim Bubblesort-Algorithmus wird die Liste (Array) wiederholt von links nach rechts durchlaufen, wobei benachbarte Elemente miteinander verglichen werden. Ist ein Element größer als das nächste (je nach Sortierkriterium), werden sie vertauscht. Dieser Vorgang wird für jedes benachbarte Paar in der Liste wiederholt, bis das Ende erreicht ist. Nachdem ein Durchlauf abgeschlossen ist, wird die Liste erneut durchlaufen, und der Vorgang wiederholt sich, bis die gesamte Liste korrekt sortiert ist. zb. [5] [1] [4] [9] [0] [8] [6]

Insertion Sort:

Insertion Sort ist ein Sortieralgorithmus, der die Liste (Array) Schritt für Schritt sortiert, indem er jedes Element in die bereits sortierte Teilliste einfügt. Dabei wird das aktuelle Element mit den vorherigen verglichen und an die richtige Position verschoben. Der Vorgang wird so lange wiederholt, bis die gesamte Liste (Array) sortiert ist. Beispiel: Angenommen, man hat ein Array [5, 2, 9, 1, 5, 6]. Der Algorithmus beginnt mit dem zweiten Element (2), und vergleicht es mit dem ersten Element (5). Da 2 kleiner ist, wird es an die erste Position verschoben, wodurch das Array [2, 5, 9, 1, 5, 6] entsteht. Dieser Prozess wird fortgesetzt, bis das Array vollständig sortiert ist: [1, 2, 5, 5, 6, 9].

Bucket Sort:

Bucket Sort ist ein nicht-vergleichbasierter Sortieralgorithmus, der eine Liste von gleichmäßig verteilten Elementen in linearer Zeit sortiert. Dieses Verfahren erfolgt dabei in drei Schritten: 1. Zuerst werden sogeanannte "Buckets" (Eimer) erstellt, in denen die Elemente der unsortierten Liste verteilt und abgelegt werden. 2. Jeder dieser Buckets wird dann mit einem anderen Sortierverfahren sortiert, wie zb. dem obigen Insertion Sort 3.Nachdem alle Buckets individuell sortiert wurden, werden die Elemente aus den Buckets in der richtigen Reihenfolge zusammengefügt, um eine neue, sortierte Liste zu erstellen.
